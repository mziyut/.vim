# .vim

## installation

### clone

```sh
$ git clone git@github.com:mziyut/.vim.git ~/.vim
```

### copy default file

```sh
$ ./bin/setup
```

### launch vim

```sh
$ vim +PlugInstall
```

## MEMO

### create dictonary file

```sh
$ ./bin/make_dict
```

### golang support
```sh
vim +:GoInstallBinaries
```

### typescript support
```sh
`npm install -g typescript-tools
```
