# .vim

## installation

### clone

```
$ git clone git@github.com:mziyut/.vim.git ~/.vim
```

### copy default file

```
$ cp ~/.vim/.vimrc.default ~/.vimrc
```

### launch vim

```
$ vim
```

## dependency

### golang
+ gocode, godef, goimports
	+ `vim +:GoInstallBinaries`

### typescript
+ typescript-tools
	+ `npm install -g typescript-tools`
